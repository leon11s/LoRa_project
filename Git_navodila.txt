Na svojmu kompu prvo povleceš najnovejšo verzijo.
$ git pull origin master

Potem programiraš svoje zadeve. In jih dodajaš.
$ git add *
$ git commit

K ti zadeve delajo in k je koda delujoca jo pushas na github.

$ git push origin master
